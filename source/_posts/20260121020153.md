#### 260120 Git Submodule 是什么？

Git 子模块允许你将一个 Git 仓库作为另一个 Git 仓库的子目录。这样你可以：

将另一个项目（比如模板、库）嵌入到你的项目中

保持子项目独立的版本控制

在主项目中引用子项目的特定版本
